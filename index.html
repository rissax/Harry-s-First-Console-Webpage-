How to create Visual Studio Code File: 1. Go on finder and create a new folder and name it whatever you want. 2. Go to Visual Studio and open your new file File. 3. Choose new folder and start coding!


To open the console in Chrome, use this keyboard shortcut: Cmd + Option + J (on a Mac) or Cmd +Shift +J (on Windows). As an alternative, you could right-click on the webpage and click "Inspect," and the developers window will appear. Click the "Console" tab in that window. You should use the console.
